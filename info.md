# Smart Heating – SONOFF TRVZB

Intelligente Heizungssteuerung für Home Assistant, speziell entwickelt für **SONOFF TRVZB Zigbee-Thermostate** via Zigbee2MQTT.

## Features

- 📅 **Wochenpläne pro Raum** — Zeitslots mit individuellen Zieltemperaturen
- 🏠 **Globale Modi** — Zuhause / Abwesend / Nacht mit einem Klick
- 🪟 **Fenster-Erkennung** — Heizung pausiert automatisch wenn Fenster offen ist
- 👤 **Anwesenheitserkennung** — Eco-Modus wenn niemand zu Hause
- 🌡️ **Außentemperatur-Kompensation** — Anpassung basierend auf Wetterdaten
- 🔥 **Boost-Modus** — Kurzes Aufheizen (30/60/90 min) per Knopfdruck
- 📊 **Seitenleisten-Panel** — Übersichtliches Dashboard in der HA-Seitenleiste

## Voraussetzungen

- Home Assistant 2023.6+
- SONOFF TRVZB Thermostate via Zigbee2MQTT
